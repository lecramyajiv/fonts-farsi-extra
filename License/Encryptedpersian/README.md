## Encrypted Persian Font
Custome persian font for encrypting and fun 

### Feature
* Vectore font file
* Base on Egyptian hieroglyphs
* Support persian/arabic code page
* TTF format

### How to use
This font is designed for print, but can also be used in the browser  
Check demo file
